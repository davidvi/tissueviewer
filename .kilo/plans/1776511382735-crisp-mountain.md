## Change 1: Toggle All Categories Button

### Location
`client/src/views/slideView.vue` — inside the "Category visibility toggles" block (around line 312–332).

### What to add
A **"Toggle all" button** (or two buttons: "Enable all" / "Disable all") placed above the per-category list. 

A single toggle button works best: when all categories are enabled it says "Disable all"; when any are disabled it says "Enable all" (or simpler: always "Toggle all categories").

**Simplest approach**: Add a single button labeled "Toggle all" that:
- If `disabledOverlayCategories` is empty → disables all (pushes all category labels into `disabledOverlayCategories`)
- Otherwise → enables all (clears `disabledOverlayCategories`)
- Calls `updateVisiblePoints()` after

### New method to add (in `<script>`)
```js
toggleAllOverlayCategories() {
  if (this.disabledOverlayCategories.length === 0) {
    // disable all
    this.disabledOverlayCategories = this.activeOverlayCategories.map(c => c.label);
  } else {
    // enable all
    this.disabledOverlayCategories = [];
  }
  this.updateVisiblePoints();
},
```

### Template change
Add above the `v-for` category list (after the `<label>Categories</label>` line):
```html
<button
  class="mb-2 px-2 py-1 text-xs rounded bg-gray-500 hover:bg-gray-400 text-white"
  @click="toggleAllOverlayCategories"
>
  {{ disabledOverlayCategories.length === 0 ? 'Disable all' : 'Enable all' }}
</button>
```

## Change 2: Hide/Show Toggle for the Left HUD

### Pattern (mirrors the right bar)
- Add a new local `data` boolean: `hudMinimal: false`
- When `hudMinimal === false`: show the full HUD panel (current behaviour, but now also add a hide button inside it)
- When `hudMinimal === true`: show a small minimized strip at the same bottom-left position with a button to re-open it

### Data property to add
```js
hudMinimal: false,
```

### Template changes

Replace the existing `#hud` block (lines 12–35) with:

```html
<!-- HUD MINIMIZED -->
<div id="hud-minimized" v-if="hudMinimal && (activatedSampleLocal && activatedSampleLocal.length > 1 || activeOverlayLegend)">
  <button class="p-2" @click="hudMinimal = false">
    <plus-circle-icon class="icon" />
  </button>
</div>

<!-- HUD FULL -->
<div id="hud" class="fixed bottom-5 left-5 max-w-60 max-h-60 overflow-y-auto bg-black bg-opacity-70 p-2 rounded-md text-white"
  v-if="!hudMinimal && (activatedSampleLocal && activatedSampleLocal.length > 1 || activeOverlayLegend)">
  
  <!-- Hide button -->
  <div class="flex justify-end mb-1">
    <button class="p-1" @click="hudMinimal = true">
      <minus-circle-icon class="icon" />
    </button>
  </div>

  <div v-if="activatedSampleLocal && activatedSampleLocal.length > 1">
    <h3 class="text-lg font-semibold mb-2">Stains</h3>
    <div v-for="channel in activatedSampleLocal" :key="channel.channel_name" class="mb-2">
      <div v-if="channel.stain != 'empty' && channel.activated">
        <div class="flex items-center space-x-2">
          <div class="w-4 h-4 rounded-full" :style="{ backgroundColor: channel.stain }"></div>
          <p>{{ channel.channel_name }}</p>
        </div>
      </div>
    </div>
  </div>
  
  <!-- Overlay Legend -->
  <div v-if="activeOverlayLegend">
    <h3 class="text-lg font-semibold mb-2 mt-3">Overlay Legend</h3>
    <div v-for="(color, label) in activeOverlayLegend" :key="label" class="mb-1">
      <div class="flex items-center space-x-2">
        <div class="w-4 h-4 rounded-full" :style="{ backgroundColor: color }"></div>
        <p class="text-sm">{{ label }}</p>
      </div>
    </div>
  </div>
</div>
```

### CSS changes
Add to `<style>`:

  position: fixed;
  bottom: 5px;
  left: 20px;
  background: rgba(0, 0, 0, 0.7);
